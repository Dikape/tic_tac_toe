# Tic-tac-toe game server
To run server on local machine, follow next step:
1. clone project
2. ```cd``` to <project_dir>
3. run command: ```docker-compose up --build -d```
4. run command: ```docker exec -it projectdir_web_1 sh```
5. run command: ```flask db upgrade```
6. run command: ```exit```
